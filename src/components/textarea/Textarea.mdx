# Textarea

A textarea allows users to input multiple lines of text.

## When to Use

Users need to enter more than a single line of text

## When not to Use

When users need to enter only one line of text, use a text input instead

## API

| Prop       | Type     | Required | Description                                              | Default |
| ---------- | -------- | -------- | -------------------------------------------------------- | ------- |
| label      | string   | false    | value of the heading label for the textarea              | ""      |
| onChange   | function | true     | function describing what to do on text area value change |         |
| id         | string   | true     | value uniquely identifying the textarea                  |         |
| isRequired | boolean  | false    | determines if the textarea is mandatory or not           | false   |

<br />
