# Header

Headers help people understand what the product or service is about while providing a consistent
look, feel, and functionality across government sites.

## When to Use

This header must appear on all public-facing online B.C. Government content and services

## When not to Use

When the product or service is not owned by the Government of British Columbia

## API

| Prop   | Type   | Required | Description                              | Default |
| ------ | ------ | -------- | ---------------------------------------- | ------- |
| header | object | true     | object with details regarding the header |         |

### Header API

The header object has its own properties that need to be present within it, as shown below:

| Prop    | Type   | Required | Description                   |
| ------- | ------ | -------- | ----------------------------- |
| name    | string | true     | label for the header          |
| history | object | true     | javascript DOM history object |

<br />

## Basic Header

Used at the top of the page.

<Story name="Basic Header" id="header--basic"></Story>

```bash
import { useHistory } from "react-router-dom";

const header = {
  name: "Header title",
  history: useHistory()
};

<Header header={header} />;
```
