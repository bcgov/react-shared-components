# Display Box

The display box is used to highlight or group together information with an optional icon.

## API

| Prop    | Type           | Required | Description                                               | Default |
| ------- | -------------- | -------- | --------------------------------------------------------- | ------- |
| styling | string         | false    | css classnames representing the styling you wish to apply | ""      |
| icon    | element        | false    | jsx element representing the icon                         | null    |
| element | string/element | true     | display element jsx for the content of the display box    |         |

<br />

## Display Box (Without Icon)

Used for content with no icon. In this case, rendering a table within the display box.

<Story name="Without Icon" id="displaybox--without-icon"></Story>

```bash
const elements = [
  { name: "BCeID", value: "bobross42" },
  {
    name: "Last Name",
    value: "Ross"
  },
  { name: "First Name", value: "Bob" },
  { name: "Email Address", value: "bob.ross@example.com" }
];

const table = <Table elements={elements} />

<DisplayBox element={table} />;
```

## Display Box (With Side by Side Content)

Used for left and right column content.

<Story
  name="With Side by Side Content"
  id="displaybox--with-side-by-side-content"
></Story>

```bash
const elements = [
  {
    name: <p style={{ width: "80%" }}>This is the content on the left of the display box. Maybe there will be other content on the right too.</p>,
    value: <p style={{ width: "80%" }}>This is the content on the right of the display box. There already is content on the left side of the box.</p>
  },
];

<DisplayBox styling="display-left-element" element={<Table elements={elements} />} />;
```

## Display Box (With File Data)

Used to display file information usually with description, icon, and fees.

<Story name="With File Data" id="displaybox--with-file-data"></Story>

```bash
const fileData = [
  {
    name: "Description:",
    value: "some file description",
    isValueBold: true
  },
  {
    name: "Statutory Fee:",
    value: "$10.00",
    isValueBold: true
  }
];

const generateTableData = () => {
  return [
    {
      name: (
        <div style={{ width: "80%" }}>
          <span>file name</span>
        </div>
      ),
      value: (
        <Table elements={fileData} />
      )
    }
  ];
};

const documentIcon = (
  <div style={{ color: "rgb(252, 186, 25)" }}>
    <MdDescription size={32} />
  </div>
);

<DisplayBox
  styling="border-background display-file"
  icon={documentIcon}
  element={<Table elementStyles={{ columnStyle: "bcgov-vertical-middle bcgov-fill-width" }} elements={generateTableData()} />}
/>
```
