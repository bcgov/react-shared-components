# Tabs

The tabs component is used to display information in an easy tab navigator.

## API

| Prop    | Type             | Required | Description                                               | Default |
| ------- | ---------------- | -------- | --------------------------------------------------------- | ------- |
| styling | string           | false    | css classnames representing the styling you wish to apply | ""      |
| tabs    | array of objects | true     | title and content of the tabs                             |         |
| height  | number           | true     | height of tabs element in pixels                          |         |

## Tabs API

The tabs array contains objects of the following structure:

| Prop       | Type           | Required | Description                            | Default |
| ---------- | -------------- | -------- | -------------------------------------- | ------- |
| title      | string         | true     | the title of a tab                     |         |
| content    | string/element | true     | the content of a tab                   |         |
| isDisabled | boolean        | false    | represents if a tab should be disabled |         |
